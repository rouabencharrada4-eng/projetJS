# NovaStore - Site E-commerce

Projet de site web e-commerce développé en HTML, CSS, JavaScript et PHP.

## Fonctionnalités prévues
- Inscription / Connexion utilisateur
- Catalogue de produits (Alimentaire, Électroménager, Cosmétiques, Vètements, Jeux, ustensiles)
- Panier d'achat
- Système de commande
- Interface administrateur

## Technologies
- Frontend : HTML5, CSS3, JavaScript
- Backend : PHP
- Base de données : MySQL
